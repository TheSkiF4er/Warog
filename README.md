# Warog

**Warog** — современный JavaScript‑фреймворк нового поколения, ориентированный на высокую производительность, минимальный рантайм и удобство разработки.

Проект разработан с максимальной уникальностью, без использования готовых фреймворков в качестве основы.

- Автор: **TheSkiF4er**
- Лицензия: **Apache 2.0**
- Версия: **1.0.0**

## Основные возможности

- Лёгкий и модульный рантайм (signals + эффекты + простой планировщик).
- Базовый компилятор для обработки `.wg.tsx` файлов.
- Клиентский рантайм с адаптерами для DOM, SSR и простого роутинга.
- Серверный адаптер для Node.js.
- Набор примитивных компонентов.
- Пример приложения `hello-warog`.

## Быстрый старт

```bash
pnpm install
pnpm build
pnpm test
```

Затем можно посмотреть пример:

```bash
cd examples/hello-warog
pnpm dev
```

## Структура репозитория

- `packages/core` — ядро рантайма.
- `packages/compiler` — зачаток компилятора.
- `packages/runtime-client` — клиентские адаптеры.
- `packages/server-adapters` — серверные адаптеры.
- `packages/components` — примитивные компоненты.
- `examples/` — примеры использования.
- `tooling/` — вспомогательные скрипты и утилиты для разработки.
